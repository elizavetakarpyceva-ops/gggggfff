# Random Quote Generator

**Автор:** [Карпычева Елизавета]
**Описание:** Простое GUI-приложение для генерации случайных цитат с фильтрацией по автору и теме, историей и сохранением данных в JSON.
**Технологии:** Python 3.x, tkinter, json

## Примеры использования

1. Запустите main.py.
2. Нажмите «Сгенерировать цитату» — появится случайная цитата.
3. Используйте фильтры для выбора по автору или теме.
4. История сохраняется и отображается в окне.
5. Кнопка «Сохранить историю» записывает историю в quotes_data.json.
6. Кнопка «Очистить историю» удаляет историю.
